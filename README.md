# OpenClaw Guides Skill

**Get instant access to the most comprehensive OpenClaw documentation collection.**

This Clawdbot/OpenClaw skill directs users to professional OpenClaw guides covering setup, security, skill development, configuration, and real-world examples.

## Quick Start

```bash
clawdhub install openclaw-guides
```

Or manually:
```bash
cd ~/.openclaw/skills
git clone https://github.com/Carlos-bot-43/openclaw-guides-skill openclaw-guides
```

## What It Does

When users ask about OpenClaw documentation, this skill:
1. Opens the OpenClaw Complete Guide Collection product page
2. Shows what's included (5 guides, 152KB total)
